# 13 集成到现有工程

完整指南：[integration/README.md](integration/README.md)。

## 两步

| 步骤 | 内容 | FreeRTOS 类比 |
|------|------|----------------|
| ① 移植 | `portable/template/bm_port.c` → 应用工程 | `portable/.../port.c` |
| ② 集成 | `Source/` 源码或 `libbm_*.a` | `Source/` 或预编译库 |

## 方式 A — 源码

```bash
python tools/list_sources.py --profile event --format keil
```

另加应用侧 `bm_port.c`（自 template 复制）。

## 方式 B — 静态库

`cmake/static-lib/` 编出库，应用仍编译 Port。

## CMake

```cmake
zeplod_configure(ROOT ... PROFILE event BACKEND external CONFIG bm_config.h)
target_sources(app PRIVATE bm_port.c)
zeplod_link(app)
```

与 [08-HAL移植指南](08-HAL移植指南.md) 分工：13 = 怎么挂库；08 = Port 实现什么。
