# 05 示例与上手路径

`examples/` 提供由浅入深的可运行程序。每个示例自带 `bm_config.h` 与 `CMakeLists.txt`，避免复制构建细节。

## 示例一览

| 示例 | 层级 | 学习重点 |
|------|------|----------|
| `ultra_blink` | Ultra | 头文件事件队列，无 `src` 链接 |
| `core_sensor` | Nano | 事件 + 内存池 + `bm_module` |
| `interrupt_demo` | Nano | SysTick/外设 ISR + ISR 侧发布事件 |
| `full_system` | Lite | 多模块、优先级、看门狗 |
| `hrt_servo_stub` | Control | 伺服三环（HRT + SRT） |
| `hrt_bms_coulomb` | Control | BMS 采样 HRT + 库仑 SRT |
| `multi_axis_sync` | Control | 多实例 + `bm_sync` |
| `multi_channel_bms` | Control | 多电芯实例模型 |

## 推荐学习顺序

```text
ultra_blink → core_sensor → interrupt_demo → full_system
                                    ↓
              hrt_servo_stub → multi_axis_sync / hrt_bms_coulomb
```

1. **core_sensor**：掌握 `bm_event_publish_copy`、`bm_module_init_all`、主循环。
2. **interrupt_demo**：理解 ISR 里只发布事件，业务在 `bm_event_process`。
3. **full_system**：看门狗与多模块协作。
4. **hrt_servo_stub**：混合域最小闭环。

## 运行方式

### 脚本（推荐）

```powershell
cd examples
.\run_single.ps1 core_sensor
.\run_all.ps1
```

```bash
./run_single.sh hrt_servo_stub
```

产物目录：`examples/build/windows/` 或 `examples/build/unix/`。成功输出 `EXAMPLE_...: PASS`。

### 手动 CMake

```bash
cmake -B build -S examples/core_sensor -DZEPLOD_BAREMETAL_DIR=../..
cmake --build build
```

QEMU 板级见各示例 `CMakeLists.txt` 中 `BOARD` 变量。

## 典型 `main.c` 结构（Lite）

```c
int main(void) {
    bm_hal_uart_init(...);
    bm_event_reset();
    bm_shell_init(&shell);
    bm_shell_register(&shell, "status", cmd_status, NULL);

    TEST_ASSERT_OK(bm_module_init_all());
    TEST_ASSERT_OK(bm_module_start_all());

    for (;;) {
        bm_event_process();
        bm_shell_poll(&shell);
        bm_wdg_feed();
    }
}
```

## 典型 `main.c` 结构（Control）

```c
int main(void) {
    bm_event_reset();
    bm_ctrl_init_all();
    bm_ctrl_start_all();
    bm_ticker_init(...);

    for (;;) {
        bm_ticker_poll();
        bm_event_process();
        bm_wdg_feed();
    }
}
```

HRT 回调在 ISR 注册，不进主循环。

## 示例约定

- 平台寄存器代码不进 `main.c`（如 `interrupt_demo/interrupt_timer.c`）。
- 异步数据优先 `bm_event_publish_copy`，避免栈指针入队。
- 共享工具在 `examples/common/`，不封装框架 API。

硬件移植文件清单：[examples/PORTING.md](../examples/PORTING.md)。
