# 06 示例与上手路径

> **本文职责**：`Demo/` 示例一览、推荐学习顺序、运行命令与硬件移植要点。  
> **数据流与 `main` 顺序** → [07-应用骨架与数据流](07-应用骨架与数据流.md)。

`Demo/` 由浅入深；各示例自带 `bm_config.h` 与 `CMakeLists.txt`。

**机制接线** → [03](03-执行域与跨域通讯.md)、[04](04-核心机制.md)  
**混合域 API 速查** → [api/](api/)（`bm_ctrl_inst`、`bm_hrt`、`bm_sync` 等）

Control 级示例的 `main` 均遵循 [07 §5.2](07-应用骨架与数据流.md#52-正确初始化顺序)：`init_all` →（可选 sync configure/arm）→ `start_all` →（可选 sync trigger）→ `ticker_init`。

## 1. 示例一览

| 示例 | 层级 | 学习重点 |
|------|------|----------|
| `ultra_blink` | Ultra | 头文件事件队列 |
| `core_sensor` | Nano | 事件 + 池 + `bm_module` |
| `interrupt_demo` | Nano | 板级 ISR → 事件入队 |
| `full_system` | Lite | 多模块、看门狗、Shell |
| `hrt_servo_stub` | Control | 三环混合域 |
| `hrt_bms_coulomb` | Control | snapshot + 库仑积分 |
| `multi_axis_sync` | Control | `bm_sync` |
| `multi_channel_bms` | Control | 多路 snapshot |

## 2. 推荐顺序

```text
ultra_blink → core_sensor → interrupt_demo → full_system
                              ↓
            hrt_servo_stub → multi_axis_sync / hrt_bms_coulomb
```

## 3. 运行

```powershell
cd Demo
.\run_single.ps1 core_sensor
.\run_all.ps1
```

```bash
./run_single.sh hrt_servo_stub
```

手动构建：

```bash
cmake -B build -S Demo/core_sensor -DZEPLOD_BAREMETAL_DIR=../..
cmake --build build
```

## 4. 目录约定

```text
Demo/
  common/           # 示例共享的轻量输出工具
  cmake/            # QEMU 示例的共享 CMake 逻辑
  examples.txt      # 运行脚本使用的示例清单
  <example>/
    main.c          # 应用流程与框架用法
    bm_config.h     # 该示例的资源配置
    CMakeLists.txt  # 声明组件和 HAL
```

## 5. 编码约定

- 寄存器/向量 ISR 放板级文件（如 `interrupt_demo/interrupt_timer.c`），不进 `main.c`。
- 异步消息用 `bm_event_publish_copy`；跨 HRT 数据用 `bm_snapshot`。

## 6. 移植到真实硬件

示例默认面向 QEMU / `native_sim`。移植到目标 MCU 时：

| 保留 | 替换 |
|------|------|
| `<example>/main.c`、`bm_config.h` | QEMU startup、链接脚本、板级 HAL |
| `common/example_support.c`（可选） | 目标板 UART / delay 实现 |
| `interrupt_demo/interrupt_timer.c` | 目标平台定时器 ISR 实现 |

各示例 `CMakeLists.txt` 或 `Makefile` 中的框架源选择为准；QEMU 共享策略在 `Demo/cmake/` 与 `qemu_example.mk`。

挂入厂商工程见 [13-集成到现有工程](13-集成到现有工程.md)；Keil / IAR 见 [18](18-Keil集成.md)、[19](19-IAR集成.md)。
